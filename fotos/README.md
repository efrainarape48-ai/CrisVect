# Galería de trabajos — Crisvect

Coloca aquí las fotos de los trabajos para mostrar en la galería del sitio web.

## Cómo agregar una foto nueva

1. Sube la imagen a esta carpeta (`fotos/`)
2. Abre el archivo `index.html`
3. Busca el array `GALLERY_ITEMS` (cerca del final del archivo)
4. Agrega una línea con este formato:

```js
{ src: "fotos/nombre-del-archivo.jpg", title: "Nombre del trabajo", cat: "diseno" },
```

## Categorías disponibles

| Código       | Categoría        |
|--------------|-----------------|
| `impresion`  | 🖨️ Impresión     |
| `diseno`     | ✏️ Diseño        |
| `stickers`   | 🏷️ Stickers      |
| `laminas`    | 📋 Láminas       |
| `arreglos`   | 💐 Arreglos      |
| `publicidad` | 📢 Publicidad    |

## Formatos recomendados

- **Formato:** JPG o PNG
- **Peso máximo:** 500 KB por foto (para que cargue rápido)
- **Tamaño:** mínimo 800px de ancho
