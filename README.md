# Crisvect — Sitio Web Oficial

**Impresión Creativa y Diseño Gráfico | Barquisimeto, Venezuela**

## Estructura del proyecto

```
CrisVect/
├── index.html        ← Página web completa
├── Logo.png          ← Logo de la marca
├── vercel.json       ← Configuración de Vercel
├── fotos/            ← AQUÍ van las fotos de la galería
│   └── README.md     ← Instrucciones para agregar fotos
└── README.md         ← Este archivo
```

## Cómo actualizar el sitio

### Agregar fotos a la galería
1. Sube la imagen a la carpeta `fotos/`
2. Edita `index.html` → busca `GALLERY_ITEMS`
3. Agrega la entrada con `src`, `title` y `cat`
4. Vercel republica automáticamente en segundos

### Actualizar el logo
Reemplaza `Logo.png` en la raíz del repositorio.
